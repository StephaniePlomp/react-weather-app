<h1>React Weather App</h1>
A simple weather app built with React, that allows you to get current weather information, based on the city you enter or giving acces to your current location. 

<h1>Demo</h1>
https://stephanieplomp.github.io/react-weather-app/


<h1>Features</h1>
<ul>
  <li>Weather forecast for any city or place</li>
  <li>Extended hourly and daily forecast</li>
  <li>Find user location</li>
  <li>One-click Celcius to Fahrenheit conversion and vice versa</li>
</ul>

<h1>APIs</h1>
<ul>
  <li> Weather data is retrieved from http://openweathermap.org/ </li>
  <li>Places autocomplete data is retrieved from Google Geocoding API</li>
</ul>
